== Description ==

### Time Ago                  

One click convert date and time to "time ago" (e.g. 1 hour ago), easy to use, just one click! translation ready (to your language).

### The Features

* Just activate the plugin to display time ago.
* Post, pages, custom post type, comments time ago support.
* Settings page to translate the plugin easily (e.g. texts: min, mins, hour, hours, etc).

== Installation ==

### Installation

1. Upload 'the-time-ago' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Now plugin is ready, but if you want to translate texts or activate comments time ago, go to Plugins menu > Time Ago.
